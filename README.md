# FlickR_Client
Flickr client will allow you to:

- Log in to your flickr account
- Like images 
- Search images
- View image data
- View image comments 
#

- using OAuthSwift to interface with flickr
- Search View Controller will load an endless amoungt of images as you scroll down.
